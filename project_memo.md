# 『Roamlust』アプリ仕様書
## アプリの目的・コンセプト
**「旅人の安全を守るオフラインファーストの危険度マップ」**
- 「Roam（歩き回る、放浪する） ＋ Lust （強い欲求）」
- REST Countries APIを元に
- 日本の外務省が発表している「海外安全情報」を元に、世界の危険度レベル（0〜4）を取得して閲覧する。
- 海外の電波がない場所でも使えるよう、基本的にオフラインでの動作を前提とする。

## 画面構成と機能（トップ画面）
アプリは1つの直感的な画面（トップ画面）に以下の3つの要素が詰まった構成です。

1. タイトル：アプリタイトル
    - 「Roamlust」と表示させる。
    - 右上に更新ボタンを設置する。

2. 上部：検索フォーム
    - 国名（例：「フランス」「ウクライナ」など）を入力して検索できる。
    - 検索すると、その国の危険度情報へ自動的にジャンプし、詳細を表示する。

3. 中央：シンプルな世界地図（メインエリア）
    - 細かい都市レベルではなく、国単位でパズルのように色分けされたシンプルな地図。
    - 外務省の基準に合わせた5色で国ごとに塗り分けられる。
        - レベル0（安全）：🟢 緑
        - レベル1（十分注意）：🟡 黄
        - レベル2（渡航自粛）：🟠 オレンジ
        - レベル3（渡航中止）：🔴 赤
        - レベル4（退避勧告）：🟣 紫
    - 世界地図を指で「ピンチイン（拡大）」「ピンチアウト（縮小）」「ググッと移動（ドラッグ）」させ、国を直接ポンとタップすることでも選択できる。

4. 下部：詳細情報表示パネル
    - 現在選択されている国の「国名」「危険度レベル」「外務省の安全概要（テキスト）」が大きく表示される。
    - 危険度レベルに応じて、パネルの背景色なども連動して変化する。
    - パネルをタップすると、画面を占める大きな詳細画面が下からスッと出てくる。
    - 国名・首都・通貨・言語・具体的な安全対策を箇条書きで表示する。（最後に選択された国にある日本大使館の住所と電話番号）

## データ管理と通信の仕様
- 通常時（オフライン）：
    アプリ内部（スマホ内）に保存されているローカルデータベースの情報を読み込んで地図や検索を表示するため、ネットがなくても完全に動作する。

- 更新時（オンライン）：
    画面右上にある「更新ボタン」を押したときだけ、インターネット経由で外務省の最新データ（RSS等）を取得しにいく。
取得した最新データでスマホ内のローカルデータベースを上書きアップデートし、地図の色や詳細テキストを最新状態に書き換える。

## 開発環境: Flutter (dart)
- flutter_svg
    画像表示**国ごとに色分けされた世界地図（SVG形式）**を画面に綺麗に描画するために使います。 
- http
    インターネット通信更新ボタンが押された時に、外務省のサーバー（RSS/XMLデータ）に最新情報を opacity（アクセス）してダウンロードするために使います。 
- shared_preferences
    オフライン保存外務省から取ってきた最新データを、スマホの内部（電源を切っても消えない場所）に簡単・安全に保存するために使います。 
- xml 、 web_feed
    XMLデータの解析（パース）外務省の文字データから「国名」「危険度」「概要」をパッと抜き出す作業が、これを使うことで数行のコードで書けるようになり、可読性が劇的に上がります。
- vector_math 
    地図の拡大・縮小・移動の制御。シンプルな世界地図とはいえ、スマホの画面だと少し小さく感じる国もあります。ユーザーが指で自由に地図を拡大してタップできるようにするために、ほぼ必須になるパッケージです。
- connectivity_plus
    インターネット接続チェック。 ユーザーが更新ボタンを押した際、今本当にネット（Wi-Fiや回線）に繋がっているかをアプリが自動で検知します。繋がっていない場合は「電波の良い場所で再度お試しください」と優しい警告を出すなど、アプリの親切度（クオリティ）が上がります。

## 情報取得
- REST Countries API
    - https://restcountries.com/v3.1/all
    - 無料で認証不要。
    - 国、首都、人口、通貨、言語、面積、国旗、タイムゾーン、地域（アジア、ヨーロッパなど）
- 海外安全情報オープンデータAPI
    - https://www.ezairyu.mofa.go.jp/html/opendata/index.html
    - 無料で認証不要。レスポンスデータ形式：XML
    - 公館コード、緊急情報も含まれる。



スマホアプリだけど、基本的にはWEBページとして作成予定。
最初のページは大きく検索フォームが来て、そこに国名（countries_masterからヒットさせる）を入力したら、その国のページに飛んで、国の諸情報やその国のwikipediaの情報が見れる。上にはタイトルとして「Roamlust」という文字を出す。タイトルの横にメニューを開けるボタンがあって、そこに国の一覧（国名と簡単な情報）ページに飛べる。もちろんその国をタップしたら国名検索で飛ぶページと同様にその国の諸情報とwikipediaのページに行く。メニューにはほかに設定としてテーマの切り替え（ライト、ダーク）とか、このアプリについて（データの出典としてwikipediaのライセンスやREST COUNTRIESの情報、Flutterのライセンス、アプリのバージョンなどがある）とか、サポート（お問い合わせ）
とかがある。国の諸情報はまずはflagsの国旗、restcountriesにあるシンプル情報、そしてwikipediaにある文章である。wikipediaの情報はすべての国にあるわけではない。wikipediaの情報が無い場合は、表示しない。ほかにも国旗の情報が無い場合もある。countries_masterとrestcountriesの数は同じなので、restcountriesの情報は表示できるはず。
