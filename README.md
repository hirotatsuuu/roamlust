# Roamlust

「Roamlust（放浪する強い欲求）」は、世界中の国々の情報を検索し、自分の「行ってみたい国リスト」を管理できる、旅好きのための探索型アプリです。

---

## 特徴

- **国検索**: 国名を検索してヒットした国の詳細情報を表示（日本語名に対応）。
- **国の一覧 & 並び替え**: 初期状態では「あいうえお順」で表示され、ランキング（人口、面積、GDP）で自由に並び替えが可能。
- **詳細情報の閲覧**: 各国の基本情報（首都、言語、通貨）やWikipediaの概要を、インターネットがないオフライン環境でも閲覧可能。
- **お気に入り機能**: 気になる国をお気に入り（ハートマーク）に登録して、自分だけの行きたい国リストを作成・管理可能。
- **完全オフライン対応**: 初回起動時にデータを一括ダウンロードした後は、キャッシュにより通信を一切行わずに全情報を閲覧可能。
- **ダークモード対応**: 端末の設定に合わせて、目の疲れを抑える美しいダークテーマ（Night Mode）に自動・手動で切り替え可能。

## インストール・実行方法

このプロジェクトは Flutter で構築されています。以下の手順でローカル環境で動作させることができます。

### 1. リポジトリのクローン
```bash
git clone [リポジトリ名]
cd roamlust
```

### 2. 依存関係のインストール
```bash
flutter pub get
```

### 3. アプリの実行
```bash
flutter run
```

## Flutter環境構築の手順

Flutterの開発を行うには、PC上に開発環境を整える必要があります。

### ステップ1: Flutter SDKのダウンロード
1. [Flutter公式サイト](https://flutter.dev)へアクセスします。
2. お使いのOS（Windows / macOS）を選択し、SDKのzipファイルをダウンロードして解凍します。
3. 解凍した `flutter` フォルダを、PCのわかりやすい場所（例: `C:\src\flutter` や `~/development/flutter`）に配置します。

### ステップ2: パス（Path）の通し方
- **Windowsの場合**: 「システム環境変数の編集」を開き、環境変数 `Path` に `C:\src\flutter\bin` を追加します。
- **macOSの場合**: 使用しているシェル（Zshなど）の設定ファイル（`~/.zshrc` など）に `export PATH="$PATH:[flutterを置いたパス]/flutter/bin"` を追記します。

### ステップ3: 依存ツールの確認
ターミナルまたはコマンドプロンプトで以下のコマンドを実行し、開発環境が整っているか確認します。
```bash
flutter doctor
```
※不足しているツール（Android StudioやXcodeなど）が表示される場合は、画面の指示メッセージに従ってインストールを行ってください。

## 重要な設定ファイル一覧

Flutterプロジェクトをカスタマイズ・管理する上で重要な主要ファイルです。

- **`pubspec.yaml`**
  アプリの名前、説明、使用する外部パッケージ（ライブラリ）や画像アセット（assets）を登録する、プロジェクトの最重要設定ファイルです。
- **`analysis_options.yaml`**
  コードを裏で自動チェックし、バグになりそうな場所や綺麗に書ける場所を波線で教えてくれる「コードの校則（静的解析ルール）」を定義するファイルです。
- **`android/app/build.gradle`**
  Androidアプリとしての最低動作バージョン（`minSdkVersion`）や、ストア公開時に必要となる世界で唯一の識別ID（`applicationId`）を設定するファイルです。
- **`ios/Runner/Info.plist`**
  iPhoneのホーム画面でアイコンの下に表示される「アプリ名（表示名）」の変更や、スマホの各種機能（カメラや位置情報など）の権限を管理する設定ファイルです。

## アプリアイコン（ランチャーアイコン）の設定方法

ホーム画面に表示されるアプリアイコン（スマホ用・Web用）は、パッケージを使用して1枚の特大画像（1024×1024）から全自動で生成しています。

1. `assets/icon.png` に 1024×1024 サイズのPNG画像を配置します。
2. ターミナルで以下のコマンドを順番に実行します。

```bash
# 依存関係を最新にする
flutter pub get

# 各OS向けのアプリアイコンを一括自動生成・配置する
dart run flutter_launcher_icons
```
